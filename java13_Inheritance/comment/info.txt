Method Overloading
- 상속과 관련 없이 하나의 클래스 내에서 일어남
- 하는 일은 동일하지만 처리하는 데이터들을 달리할 때 사용


<Rule>
1. 메소드 이름 같음
2. 메소드 역할 같음
3. 메소드 인자 다름
4. return 값의 데이터 형식은 달라도 됨


생성자 Overloading
- 하나의 클래스 안에서 생성자를 여러개 사용할 수 있다
- 객체 생성 시 모든 필드의 값이 온전히 채워지지 않더라도 객체는 생성될 수 있어야 한다

<Rule>
1. 메소드 오버라이딩과 동일함
2. 생성자에 return 값은 없으므로 4번 생략


Method Overriding
- 상속 관계에 있는 2개의 클래스에서 작용하는 기능
- 자식이 부모의 메소드를 상속 받아 자식 객체에 맞게 변경하여 사용

<Rule>
1. 메소드 이름 같음
2. 메소드 인자 같음
3. 메소드 역할 다름
4. return 데이터 형식 같음
